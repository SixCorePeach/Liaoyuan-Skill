## 达尔文站在哪？

达尔文这个名字暗含了一个世界观：

```
自然选择 → 被动适应 → 适者生存
```

对于skill 优化：**让 rubric 当自然环境，skill 在评分压力下被动淘汰退步的、保留进步的。** Skill 没有主动性，它只是"被选择"的对象。

达尔文的天花板——**他的革命是解释世界的革命，不是改造世界的革命。**

---

## 我们站在哪？

马克思说的那句话，才是我们的站位：

> "哲学家们只是用不同的方式解释世界，而问题在于改变世界。"

达尔文 = 解释世界（被动发现规律）
燎原 = **改造世界**（主动用实践改造理论、用理论改造实践）

skill的优化：**不是被选择，是去点燃。**

---

**燎原·skill**

> 星星之火，可以燎原。

### 为什么比达尔文高级

| | 达尔文 | 燎原 |
|:--|:------|:-----|
| 哲学根基 | 自然选择——被动适应 | 星火燎原——主动改造 |
| 世界观 | 物竞天择，适者生存 | 实践检验，矛盾驱动 |
| Skill的地位 | 被环境选择的客体 | 主动点燃变革的主体 |
| 进步方式 | 渐变积累（mutation → selection） | 质变飞跃（量变→质变→螺旋上升） |
| 终极隐喻 | 生存 = 最低标准 | 燎原 = 最高境界 |
| 历史分量 | 解释了生命从何而来 | 预言了革命如何胜利 |

**达尔文告诉你：活下来的就是好的。**
**燎原告诉你：一点火种，足以改天换地。**

### 燎原的每一层含义

```
星火 = 一个bug的修复，看似微小
燎原 = 整个skill质量的系统性跃升

星火 = 一次实践验证，看似局部
燎原 = rubric被实践修正后的全局进化

星火 = 一个维度的改进，看似单一
燎原 = 矛盾转化后主要矛盾的系统性解决
```

这不是渐进改良，是**从量变到质变**——每个小改进都是星火，但星火不是目的，燎原才是。达尔文只看到了星火（变异），没看到燎原（革命）。我们看到的是：**星火必然燎原，因为矛盾驱动的方向不可逆。**
